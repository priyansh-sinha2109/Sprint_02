# Cash Flow - Expense Tracker

## Overview

Cash Flow is a personal finance tracking application built using Vanilla JavaScript, HTML, and CSS. Users can set their salary, track expenses, monitor remaining balance, generate reports, and visualize spending through charts.

## Features

### Phase 1 - Base MVP

- Set Total Salary
- Add Expense Name and Expense Amount
- Dynamic DOM Updates
- Real-Time Remaining Balance Calculation
- Input Validation
- Vanilla JavaScript Implementation

### Phase 2 - Data Persistence & Visualization

- LocalStorage Integration
- Expense Deletion
- Expense Editing
- Persistent Data After Reload
- Dynamic Pie Chart using Chart.js
- Automatic Balance Recalculation

### Phase 3 - Advanced Features

- PDF Report Generation using jsPDF
- Low Balance Warning Alert
- Currency Conversion (INR, USD, EUR, GBP)
- Real-Time Dashboard Updates

## Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage
- jsPDF
- Frankfurter Currency API
